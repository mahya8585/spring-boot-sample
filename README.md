# Legacy Spring Boot 2.3 Application Workshop

## Description

This repository contains a comprehensive workshop for migrating legacy Spring Boot 2.3 applications to modern architecture using GitHub Copilot and contemporary development practices. The workshop includes hands-on materials, tutorials, exercises, and reference implementations for a complete modernization journey.
